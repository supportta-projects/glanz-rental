# Glanz Rental Design System
## Premium SaaS Design Standards

### Color Palette
- **Primary**: `#0b63ff` (Blue)
- **Primary Hover**: `#0a5ce6`
- **Primary Foreground**: `#ffffff`
- **Secondary**: `#f1f5f9` (Light Gray)
- **Background**: `#f7f9fb` (Very Light Gray)
- **Foreground**: `#0f1724` (Dark Gray)
- **Success**: `#0f9d58` (Green)
- **Warning**: `#f59e0b` (Orange)
- **Destructive**: `#ef4444` (Red)
- **Muted**: `#6b7280` (Gray)

### Typography
- **Font Family**: Geist Sans (system fallback)
- **Headings**: Bold, Dark Gray (`#0f1724`)
- **Body**: Regular, Dark Gray
- **Muted Text**: `#6b7280`
- **Font Sizes**:
  - Page Title: `text-2xl md:text-3xl font-bold`
  - Section Title: `text-xl font-bold`
  - Body: `text-sm` (14px)
  - Small: `text-xs` (12px)

### Spacing
- **Container Padding**: `px-4 md:px-6 py-4`
- **Card Padding**: `p-4 md:p-6`
- **Gap Between Elements**: `gap-2` to `gap-4`
- **Section Spacing**: `space-y-4` to `space-y-6`

### Buttons
- **Primary Button**: 
  - Height: `h-7` (small), `h-8` (medium), `h-9` (large)
  - Background: `bg-[#0b63ff]`
  - Hover: `hover:bg-[#0a5ce6]`
  - Text: `text-white text-xs font-medium`
  - Padding: `px-2` to `px-3`
  - Border Radius: `rounded-lg`

- **Outline Button**:
  - Border: `border-[#0b63ff] text-[#0b63ff]`
  - Hover: `hover:bg-[#0b63ff] hover:text-white`

### Inputs
- **Height**: `h-8` (standard)
- **Border**: `border-gray-200`
- **Focus**: `focus:border-[#0b63ff] focus:ring-1 focus:ring-[#0b63ff]`
- **Border Radius**: `rounded-lg`
- **Placeholder**: `placeholder:text-gray-400`

### Cards
- **Background**: `bg-white`
- **Border**: `border border-gray-200`
- **Border Radius**: `rounded-lg`
- **Shadow**: `shadow-sm`
- **Padding**: `p-4` to `p-6`

### Tables
- **Header Background**: `bg-[#f1f5f9]`
- **Row Hover**: `hover:bg-zinc-50`
- **Border**: `border-gray-200`
- **Cell Padding**: `px-4 py-3`

### Badges
- **Default**: `bg-gray-100 text-gray-700`
- **Primary**: `bg-[#0b63ff] text-white`
- **Success**: `bg-green-100 text-green-700`
- **Warning**: `bg-orange-100 text-orange-700`
- **Danger**: `bg-red-100 text-red-700`

### Icons
- **Size**: `h-3 w-3` (small), `h-4 w-4` (medium), `h-5 w-5` (large)
- **Color**: Inherit from parent or use semantic colors

### Layout
- **Sidebar Width**: `250px` (desktop)
- **Content Margin**: `md:ml-[250px]`
- **Max Content Width**: `max-w-7xl` (when needed)
- **Container**: Full width with padding

### Responsive Breakpoints
- **Mobile**: `< 768px`
- **Desktop**: `>= 768px`

### Animation & Transitions
- **Duration**: `transition-all duration-200`
- **Hover Effects**: Smooth color transitions
- **Loading**: `animate-spin` for spinners
- **Skeleton**: `animate-pulse`

### Status Colors
- **Active/Ongoing**: `#f59e0b` (Orange)
- **Completed/Returned**: `#3b82f6` (Blue)
- **Late**: `#ef4444` (Red)
- **Scheduled**: `#9ca3af` (Gray)
- **Cancelled**: `#6b7280` (Gray)
